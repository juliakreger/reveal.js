A little background!

OpenStack could be described as a conglomerate of projects. Some official, some not so much. Many projects have specific focus areas, and may have dependencies. A cross project feature is a feature that involves two or more projects in concert.

So you want to add a feature?

Tossing code at the wall doesn't always help!

But getting consensus about tossing code upon the wall could...

The Beginning!

I was in a conference room in Vancouver, BC and desire from vendors and community was for cinder integration to exist. It seemed simple to me, it seemed feasible, do-able. So why not?!?
